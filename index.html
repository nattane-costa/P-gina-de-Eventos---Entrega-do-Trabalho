Sobre

A disciplina de Programação para Web I tem como objetivo capacitar o aluno a compreender e desenvolver o front-end de sistemas web.

Programação

Ementa da Disciplina

Nesta disciplina, abordaremos os seguintes tópicos:

Servidores WEB. Protocolo HTTP. HTML5. Cascading Style Sheets (CSS). Layouts responsivos. JavaScript. Gerenciadores de conteúdo. Uso de frameworks atuais

Dias de Aula

Quarta: 7h

Sexta: 9h10

Data de Entrega

Entrega do Trabalho (versão 1.0): 30/10/2024

Perguntas Frequentes

A entrega de trabalho é um evento no qual os alunos apresentam seus projetos finais, no formato de stands, com o objetivo de compartilhar conhecimento e receber feedback. Além do evento em si, o aluno deverá enviar o link do github para a professora.

Apenas os alunos do 4º período de SI matriculados na disciplina de Web I podem participar.

Não, se você está matriculado na disciplina você está automaticamente inscrito no evento.

A versão 1.0 é a versão inicial dos seus projetos construídos apenas com HTML, CSS e JavaScript. No final da disciplina você deverá realizar a entrega do trabalho final versão 2.0, que contará com a implementação dos mesmos projetos utilizando Angular.
